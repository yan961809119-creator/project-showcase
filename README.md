# 澄野 Studio 项目展示网站

面向客户与合作方的中文单页项目展示网站，采用 Next.js、TypeScript 和 Tailwind CSS 构建。

## 本地运行

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 内容替换

首版示例内容集中在 `app/page.tsx`。请替换品牌名称、项目介绍、指标、邮箱及社交链接。SEO 信息位于 `app/layout.tsx`。

## 部署

将项目推送至 GitHub 后，在 Vercel 中导入仓库即可部署。Vercel Analytics 已接入，部署后会自动开始收集基础访问数据。
